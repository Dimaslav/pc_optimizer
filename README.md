PC Optimizer
Безопасная утилита для анализа и очистки Windows с графическим интерфейсом на PyQt5.
Программа помогает находить временные файлы, кэши браузеров и приложений, большие файлы, дубликаты, а также управлять процессами и элементами автозагрузки.

Важно: перед очисткой приложение показывает список найденных файлов. Удаляются только отмеченные пользователем элементы.


Возможности
Очистка

временные файлы пользователя;
кэш миниатюр Windows;
кэши Chrome, Edge, Brave и Firefox;
игровые кэши Discord, Epic Games, NVIDIA, Battle.net и Direct3D;
список недавних документов;
корзина Windows;
кэш DNS;
настройка минимального возраста удаляемых файлов;
предварительный просмотр перед удалением;
выбор отдельных файлов;
экспорт списка в CSV;
JSON-отчёты о выполненных операциях;
отмена длительной операции.

Анализ файлов

поиск больших файлов;
настройка минимального размера;
поиск дубликатов;
двухэтапное сравнение дубликатов:

размер и частичный SHA-256;
полный SHA-256;


открытие расположения найденного файла двойным щелчком.

Автозагрузка

просмотр записей HKCU\...\Run;
просмотр 32- и 64-битных записей HKLM\...\Run;
просмотр пользовательской и общей папок Startup;
безопасное отключение элементов;
резервное копирование отключённых записей;
восстановление элементов автозагрузки.

Система

оценка состояния компьютера;
отображение нагрузки CPU;
отображение использования оперативной памяти;
отображение свободного места на системном диске;
список запущенных процессов;
завершение выбранных процессов;
защита системных процессов от случайного завершения;
блокировка запуска второго экземпляра программы;
ротация журналов.


Требования

Windows 10 или Windows 11;
Python 3.9 или новее;
PyQt5;
psutil.

Приложение предназначено только для Windows.

Установка
1. Установите Python
Скачайте Python с официального сайта:
https://www.python.org/downloads/
Во время установки отметьте пункт:
Add Python to PATH

Проверьте установку:
python --version

2. Перейдите в папку проекта
cd C:\путь\к\pc_optimizer

Например:
cd C:\Users\User\Desktop\pc_optimizer

3. Создайте виртуальное окружение
Рекомендуется запускать проект в отдельном окружении:
python -m venv .venv

Активируйте его:
.venv\Scripts\activate

4. Установите зависимости
python -m pip install --upgrade pip
python -m pip install PyQt5 psutil

Можно создать файл requirements.txt:
PyQt5>=5.15.10
psutil>=5.9.8

После этого зависимости устанавливаются командой:
python -m pip install -r requirements.txt


Проверка файлов
Перед первым запуском проверьте синтаксис всех модулей:
python -m py_compile core.py operations.py workers.py ui.py main.py

Если команда ничего не вывела, синтаксических ошибок не обнаружено.

Запуск
Запускайте программу через main.py:
python main.py

Не запускайте отдельно:
core.py
operations.py
workers.py
ui.py

Они являются внутренними модулями приложения.

Использование
Dashboard
На главной вкладке отображаются:

оценка состояния системы;
загрузка CPU;
загрузка оперативной памяти;
свободное место на системном диске;
дата последней очистки;
объём освобождённого места;
системные предупреждения;
состояние текущей фоновой операции.

Очистка

Откройте вкладку «Очистка».
Отметьте нужные категории.
Укажите минимальный возраст файлов.
Нажмите «Найти файлы».
Просмотрите результаты.
Снимите отметки с файлов, которые не нужно удалять.
При необходимости экспортируйте список в CSV.
Нажмите «Удалить отмеченное».
Подтвердите операцию.


Файлы удаляются напрямую, без перемещения в корзину.

Значение возраста:

0 часов — учитывать все найденные файлы;
24 часа — учитывать файлы старше одного дня;
72 часа — учитывать файлы старше трёх дней;
168 часов — учитывать файлы старше одной недели.

Корзина
Кнопка «Очистить корзину» показывает приблизительный размер содержимого и запрашивает подтверждение.
Очистка применяется к корзинам доступных дисков.
DNS
Кнопка «Очистить DNS» выполняет:
ipconfig /flushdns

Эта операция не освобождает дисковое пространство, но сбрасывает локальный DNS-кэш Windows.
Большие файлы

Откройте вкладку «Файлы».
Укажите одну или несколько папок.
Несколько путей разделяйте точкой с запятой:

C:\Users\User\Downloads; D:\Video; D:\Games


Установите минимальный размер.
Нажмите «Найти большие файлы».
Дважды щёлкните по результату, чтобы открыть его расположение.

Программа не удаляет большие файлы автоматически.
Дубликаты

Укажите папки для поиска.
Установите минимальный размер файла.
Нажмите «Найти дубликаты».
Просмотрите найденные группы.
Откройте расположение файлов двойным щелчком.
Удаляйте ненужные копии вручную через Проводник.

Файлы считаются дубликатами только после совпадения полного SHA-256.

Перед ручным удалением проверьте содержимое файлов. Одинаковый файл может быть нужен нескольким программам или проектам.

Автозагрузка

Откройте вкладку «Автозагрузка».
Выберите элемент.
Нажмите «Отключить выбранное».
Подтвердите действие.

Для записи реестра сохраняются:

имя;
значение;
тип значения;
корень реестра;
32- или 64-битное представление.

Файл из папки Startup перемещается в резервное хранилище:
%LOCALAPPDATA%\PCOptimizer\startup_backups

Для восстановления:

Нажмите «Восстановить из резервной копии».
Выберите элемент.
Подтвердите восстановление.

Процессы
На вкладке «Процессы» можно:

обновить список;
посмотреть PID;
посмотреть использование памяти;
посмотреть статус;
посмотреть пользователя;
завершить выбранный процесс.


Завершение процесса может привести к потере несохранённых данных.

Программа запрещает завершать:

PID 0;
PID 4;
собственный процесс PC Optimizer.


Права администратора
Для основной пользовательской очистки права администратора обычно не нужны.
Повышенные права могут потребоваться для:

отключения записей автозагрузки из HKLM;
завершения системных процессов;
доступа к защищённым каталогам.

Запуск через PowerShell с повышением прав:
Start-Process python -ArgumentList "main.py" -Verb RunAs

Или откройте командную строку от имени администратора и выполните:
python main.py


Не рекомендуется постоянно запускать программу с правами администратора без необходимости.


Меры безопасности
В приложении предусмотрены следующие ограничения:

запрещено удаление корня диска;
запрещено удаление корня категории очистки;
запрещено удаление базового каталога профиля пользователя;
запрещено удаление каталога Windows целиком;
запрещено удаление каталогов Program Files целиком;
очистка не проходит через символические ссылки;
очистка не проходит через junction;
очистка не проходит через другие reparse points;
удаление выполняется только внутри разрешённого корня;
перед удалением показывается предварительный список;
длительные операции выполняются в фоновом потоке;
можно запросить отмену операции;
одновременно выполняется только одна тяжёлая операция;
второй экземпляр программы блокируется;
результаты операций записываются в отчёты.

Программа намеренно не выполняет:

прямое удаление Windows.old;
полную очистку C:\Windows\Logs;
удаление системных каталогов;
автоматическое удаление найденных больших файлов;
автоматическое удаление дубликатов;
изменение типа запуска служб;
автоматическое скачивание и установку обновлений.

подробности обработки файлов.


Сборка EXE
1. Установите PyInstaller
python -m pip install pyinstaller

2. Выполните сборку
pyinstaller --noconfirm --clean --windowed --name "PC Optimizer" main.py

Готовый файл будет создан в:
dist\PC Optimizer\PC Optimizer.exe

Сборка одним EXE
pyinstaller --noconfirm --clean --onefile --windowed --name "PC Optimizer" main.py

Результат:
dist\PC Optimizer.exe

Сборка с иконкой
pyinstaller --noconfirm --clean --onefile --windowed --name "PC Optimizer" --icon app.ico main.py


Для стабильной работы и более быстрого запуска предпочтительнее обычная сборка без --onefile.

Сборка с запросом прав администратора
Если действительно требуется постоянное повышение прав:
pyinstaller --noconfirm --clean --windowed --uac-admin --name "PC Optimizer" main.py

Постоянное использование --uac-admin не рекомендуется.

Устранение проблем
ModuleNotFoundError: No module named 'PyQt5'
Установите PyQt5 в тот Python, которым запускается программа:
python -m pip install PyQt5

ModuleNotFoundError: No module named 'psutil'
python -m pip install psutil

Ошибка импорта ProgressBar
В ui.py должен импортироваться:
QProgressBar

В файле не должно быть:
ProgressBar
QProgressBar = ProgressBar

SyntaxError рядом с <span>
В код случайно попала HTML-разметка. Найдите и удалите конструкции:
<span
</span>
footnote-wrapper

После этого проверьте файлы:
python -m py_compile core.py operations.py workers.py ui.py main.py

Программа запускается и сразу закрывается
Запускайте её из командной строки, чтобы увидеть ошибку:
cd C:\путь\к\pc_optimizer
python main.py

Также проверьте журнал:
%LOCALAPPDATA%\PCOptimizer\logs\pc_optimizer.log

Не удаётся отключить автозагрузку
Если запись находится в HKLM, запустите программу от имени администратора.
Не удаётся удалить некоторые файлы
Возможные причины:

файл используется запущенной программой;
браузер или игровой клиент не закрыт;
недостаточно прав;
файл заблокирован Windows;
путь является ссылкой или reparse point;
файл уже был удалён другой программой.

Закройте соответствующее приложение и повторите сканирование.
Отмена выполняется не мгновенно
Отмена проверяется между файлами и блоками хеширования. Если обрабатывается большой файл, завершение текущего шага может занять некоторое время.
Второй экземпляр не запускается
Это нормальное поведение. Приложение использует Windows mutex и разрешает работу только одного экземпляра.
Если программа аварийно завершилась, mutex автоматически освобождается операционной системой.
Отображаются иероглифы в CSV
Файл сохраняется в кодировке UTF-8 с BOM. Для корректного открытия используйте современную версию Excel или импортируйте CSV с кодировкой UTF-8 и разделителем ;.

Сброс настроек
Закройте приложение и удалите файл:
%LOCALAPPDATA%\PCOptimizer\config\settings.json

При следующем запуске будут созданы настройки по умолчанию.
Не удаляйте startup_backups.json, если планируете восстанавливать отключённые элементы автозагрузки.

Полное удаление программы

Закройте PC Optimizer.
Удалите папку с исходным кодом или собранной программой.
При необходимости удалите пользовательские данные:

%LOCALAPPDATA%\PCOptimizer


Удаление этого каталога уничтожит журналы, отчёты и резервные копии автозагрузки.
