# PC Optimizer
Простая утилита для анализа и безопасной очистки Windows, написанная на Python и PyQt5.
## Возможности
- очистка временных файлов;
- очистка кэша браузеров и игровых приложений;
- очистка корзины и DNS-кэша;
- предварительный просмотр файлов перед удалением;
- поиск больших файлов;
- поиск дубликатов по SHA-256;
- управление автозагрузкой с резервным копированием;
- просмотр и завершение процессов;
- отображение загрузки CPU, памяти и диска;
- отмена длительных операций;
- сохранение отчётов и логов.
## Требования
- Windows 10/11;
- Python 3.9 или новее.
## Установка
```cmd
git clone https://github.com/USERNAME/pc-optimizer.git
cd pc-optimizer
python -m pip install -r requirements.txt
Содержимое requirements.txt:
PyQt5>=5.15.10
psutil>=5.9.8
Запуск
python main.py
Структура проекта
pc-optimizer/
├── core.py
├── operations.py
├── workers.py
├── ui.py
├── main.py
└── requirements.txt
Сборка EXE
python -m pip install pyinstaller
pyinstaller --noconfirm --clean --onefile --windowed --name "PC Optimizer" main.py
Готовый файл появится в папке dist.
Безопасность
Программа:
 * показывает файлы перед удалением;
 * не удаляет системные каталоги;
 * не проходит через символические ссылки и junction;
 * не удаляет большие файлы и дубликаты автоматически;
 * создаёт резервные копии отключённых элементов автозагрузки.
> Файлы, выбранные на вкладке очистки, удаляются без перемещения в корзину. Проверяйте список перед подтверждением.
Данные программы
Настройки, логи и отчёты хранятся в:
%LOCALAPPDATA%\PCOptimizer
