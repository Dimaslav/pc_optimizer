# 🧹 PC Optimizer

Безопасная утилита для анализа и очистки Windows с графическим интерфейсом на **Python + PyQt5**.

PC Optimizer находит временные файлы, кэши приложений, большие файлы и дубликаты, а также позволяет управлять процессами и элементами автозагрузки.

> [!IMPORTANT]
> Перед удалением программа показывает найденные файлы. Удаляются только элементы, выбранные пользователем.

## ✨ Возможности

### Безопасная очистка

- временные файлы пользователя;
- кэш миниатюр Windows;
- кэши Chrome, Edge, Brave и Firefox;
- кэши Discord, Epic Games, NVIDIA, Battle.net и Direct3D;
- список недавних документов;
- корзина Windows;
- DNS-кэш;
- фильтрация файлов по возрасту;
- предварительный просмотр перед удалением;
- выбор отдельных файлов;
- экспорт результатов в CSV;
- сохранение отчётов в JSON;
- отмена длительных операций.

### Анализ файлов

- поиск больших файлов;
- настраиваемый минимальный размер;
- поиск дубликатов;
- частичное и полное сравнение SHA-256;
- отображение потенциально освобождаемого места;
- открытие расположения файла двойным щелчком.

### Управление автозагрузкой

- записи `HKCU\...\Run`;
- 32- и 64-битные записи `HKLM\...\Run`;
- пользовательская и общая папки Startup;
- безопасное отключение элементов;
- резервное копирование отключённых записей;
- восстановление автозагрузки через интерфейс.

### Мониторинг системы

- оценка состояния компьютера;
- загрузка CPU;
- использование оперативной памяти;
- свободное место на системном диске;
- просмотр запущенных процессов;
- завершение выбранных процессов;
- защита критических PID от случайного завершения;
- блокировка запуска второго экземпляра.

## 🛡️ Безопасность

PC Optimizer намеренно ограничивает потенциально опасные операции:

- не удаляет корни дисков;
- не удаляет корневые системные каталоги;
- не проходит через symlink, junction и другие reparse points;
- проверяет принадлежность файла разрешённой папке;
- показывает файлы до удаления;
- не удаляет `Windows.old` напрямую;
- не очищает целиком `C:\Windows\Logs`;
- не удаляет большие файлы и дубликаты автоматически;
- сохраняет резервные копии отключённой автозагрузки;
- выполняет тяжёлые операции в фоновом потоке;
- разрешает только одну тяжёлую операцию одновременно.

> [!WARNING]
> Файлы, отмеченные на вкладке очистки, удаляются напрямую, без перемещения в корзину. Перед подтверждением внимательно проверьте список.
